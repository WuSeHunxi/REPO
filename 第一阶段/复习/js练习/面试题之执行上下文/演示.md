foo函数的执行上下文建立过程:

{
    vo:{
        a:指向函数,
        b:4,
        arguments: {....},
        j:undefined,
        bar:undefined
    }
}

**创建vo对象**

1. 确定形参值
2. 确定函数中所有的函数字面量声明
   1. 该函数必须是字面量声明，字面量声明提取到vo后，可以认为该声明失效
   2. 如果当前vo中出现同名属性，直接覆盖
3. 确定函数中所有的变量声明（var），将其提取到上下文中，值为undefined
   1. 如果当前vo中出现同名属性，忽略
